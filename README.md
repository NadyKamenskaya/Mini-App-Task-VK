## Mini-App-Task-VK

Test task for an internship at VK Company

## Get started

### Clone repository

```bash
git clone git@github.com:NadyKamenskaya/Mini-App-Task-VK.git
```

### Install dependencies

```bash
npm install
```

### App builder

```bash
npm run build
```

### Running the application

```bash
npm start
```
